# v1.13 Directional transitions (Phase 3) + 모든 새 Trigger 타입 — Implementation Plan

> **For agentic workers:** Use superpowers:subagent-driven-development.

**Goal:** v0.11.0 기준의 SimpleTransition (3 types) + 4 trigger를 확장해서:
1. **DirectionalTransition** 5종 × 4방향 × matchLayers 노출 (Phase 3)
2. **새 Trigger 7종** 노출: ON_DRAG, MOUSE_UP, MOUSE_DOWN, MOUSE_ENTER, MOUSE_LEAVE, ON_KEY_DOWN, ON_MEDIA_HIT, ON_MEDIA_END

**Architecture:** TransitionInput / TriggerInput Zod union 양쪽 모두 string shortcut + 객체 variant를 동시 허용. `buildTransition`은 directional `type` 5종을 simple과 분기해서 emit. `buildTrigger`는 string 1-인자 + object 다-variant switch로 재작성. 기존 top-level `afterTimeoutSeconds`는 string `"AFTER_TIMEOUT"`과만 결합 가능 (backward compat); 새 object `{type:"AFTER_TIMEOUT", timeout}` 형태는 self-contained.

**Source spec:** `docs/superpowers/specs/2026-05-20-v1.13-directional-and-triggers-design.md`

- [ ] **Step 4: Rewrite `buildTrigger`**

```ts
export function buildTrigger(
  input: TriggerInput,
  legacyAfterTimeoutSeconds?: number
): TriggerShape {
  if (typeof input === "string") {
    if (input === "AFTER_TIMEOUT") {
      if (legacyAfterTimeoutSeconds === undefined) {
        throw new Error("buildTrigger: afterTimeoutSeconds is required when name is AFTER_TIMEOUT");
      }
      return { type: "AFTER_TIMEOUT", timeout: legacyAfterTimeoutSeconds };
    }
    return { type: input };
  }
  switch (input.type) {
    case "ON_CLICK":
    case "ON_HOVER":
    case "ON_PRESS":
    case "ON_DRAG":
    case "ON_MEDIA_END":
      return { type: input.type };
    case "AFTER_TIMEOUT":
      return { type: "AFTER_TIMEOUT", timeout: input.timeout };
    case "MOUSE_UP":
    case "MOUSE_DOWN":
      return { type: input.type, delay: input.delay ?? 0 };
    case "MOUSE_ENTER":
    case "MOUSE_LEAVE":
      return {
        type: input.type,
        delay: input.delay ?? 0,
        deprecatedVersion: input.deprecatedVersion ?? false,
      };
    case "ON_KEY_DOWN":
      return { type: "ON_KEY_DOWN", device: input.device, keyCodes: input.keyCodes };
    case "ON_MEDIA_HIT":
      return { type: "ON_MEDIA_HIT", mediaHitTime: input.mediaHitTime };
  }
}
```

- [ ] **Step 4: Update `buildTransition`**

기존 함수 본문 변경 — `input === "INSTANT"` 가지 + `DISSOLVE`/`SMART_ANIMATE` 가지 그대로 유지. 그 뒤 객체 가지에서 directional / simple 분기:

```ts
export function buildTransition(input: TransitionInput): TransitionShape {
  if (input === "INSTANT") return null;
  if (input === "DISSOLVE" || input === "SMART_ANIMATE") {
    return { type: input, duration: 0.3, easing: { type: "EASE_OUT" } };
  }
  // Object form — discriminate by type
  if (
    input.type === "MOVE_IN" || input.type === "MOVE_OUT" ||
    input.type === "PUSH"    || input.type === "SLIDE_IN" ||
    input.type === "SLIDE_OUT"
  ) {
    return {
      type: input.type,
      direction: input.direction,
      matchLayers: input.matchLayers ?? false,
      duration: input.duration ?? 0.3,
      easing: resolveEasing(input.easing),
    };
  }
  // SimpleTransitionInput
  return {
    type: input.type,
    duration: input.duration ?? 0.3,
    easing: resolveEasing(input.easing),
  };
}
```
