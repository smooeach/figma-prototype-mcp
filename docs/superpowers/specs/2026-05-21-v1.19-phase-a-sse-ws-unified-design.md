# figma-prototype-mcp v1.19 — Phase A: Transport+Network 통합 설계 문서

- 작성일: 2026-05-21
- 상태: 초안 (사용자 승인 완료)
- 기반 버전: v0.17.0
- 시리즈: v2 아키텍처의 **첫 번째 phase**. v1.20 (Phase B: `proto_*` + motion presets), v1.21 (Phase C: history stack) 이후로 분리.
- 디자이너 노트: 이 release는 "엔진 교체" 같은 작업이라 디자이너 입장에서 동작은 같아 보여야 함. 단지 더 빠르고, 켜둘 수 있고, 24시간 돌아감.

## 1. 목적

v0.17.0 기준 구조 = 3개 별도 프로세스 (Stdio MCP 자식 + WebSocket relay on port 3055 + Figma plugin). 매 세션 새로 시작, 디자이너가 3개 터미널 띄워야 함.

v1.19 (Phase A)는 이걸 **1개 Express 프로세스**로 통합:
- Claude는 SSE(HTTP)로 접속 (stdio 폐기)
- Figma plugin은 같은 Express의 WebSocket으로 접속
- 동일 포트 (3000) 공유, 한 번 켜두면 24/7 동작
- 기능 변화 0 — 10개 tool 그대로, 디자이너 자연어 입력 100% 그대로

목적은 **운영 단순화 + 24/7 상시 구동** 기반 마련. v1.20/v1.21이 이 기반 위에 새 기능 얹음.

## 2. Spec-stage 호환성 점검

- `@modelcontextprotocol/sdk` ^1.0.0 — SSE transport 제공 (`SSEServerTransport`). v1.0 SDK가 SSE 지원 확인.
- `express` ^4 — Node 20+ ESM 호환 (`@types/express` devDep 추가).
- `ws` ^8 — 이미 설치됨. `noServer: true` 모드로 Express HTTP server의 `upgrade` 이벤트 핸들링.
- ESM (`"type": "module"`) 유지. tsconfig.json은 ES2022 + ESNext 모듈 그대로.

**Task 1 bypass-probe 의무**: SSE+WS 같은 포트 공유가 SDK + express + ws 조합으로 실제 작동하는 minimal POC를 코드 변경 전에 확인. Figma 관련 probe는 아니고 backend 인프라 probe.

## 3. 스코프

### Phase A 포함

- Stdio MCP → Express + SSE on port 3000 (env `PORT`로 override 가능).
- WebSocket을 같은 Express에 `/ws` path로 통합 (port 공유).
- Standalone `src/socket.ts` 폐기 (functionality 흡수).
- `PluginSession` 클래스 — single-active plugin 관리 (가장 최근 연결 == active, 이전 연결은 정중히 disconnect).
- 3-second timeout guard — plugin 미연결 시 tool 호출에서 최대 3s 대기 후 "피그마 플러그인 연결을 확인해주세요" 에러.

- Channel-based pub/sub 프로토콜 제거 → 직접 command/response 메시지로 단순화.

- Claude Desktop/Code 설정 변경 안내 (stdio → SSE URL).

## 6. WebSocket 프로토콜 변경

### Before (channel-based)

```json
// Plugin → server
{ "type": "join", "channel": "test1", "id": "mcp-join" }

// Server → plugin (system)
{ "type": "system", "message": "Connected to channel: test1" }

// MCP server → plugin (via relay broadcast)
{ "type": "message", "channel": "test1",
  "message": { "id": "uuid", "type": "command", "command": "CREATE_REACTIONS", "params": {...} } }

// Plugin → MCP server (via relay broadcast)
{ "type": "broadcast", "channel": "test1", "sender": "...",
  "message": { "id": "uuid", "type": "response", "status": "ok", "result": ... } }
```

### After (single connection direct)

```json
// Server → plugin on connect
{ "type": "ready" }

// Server → plugin (command)
{ "type": "command", "id": "uuid", "command": "CREATE_REACTIONS", "params": {...} }

// Plugin → server (response)
{ "type": "response", "id": "uuid", "status": "ok", "result": ... }

// Plugin → server (response, error case)
{ "type": "response", "id": "uuid", "status": "error", "error": { "code": "...", "message": "..." } }
```

Channel 필드 / join handshake 제거. id는 server가 생성하고 응답 매칭에 사용.

## 7. PluginSession (sessions.ts)

```ts
import type { WebSocket } from "ws";
import { randomUUID } from "node:crypto";

interface Pending {
  resolve: (v: unknown) => void;
  reject: (e: Error) => void;
  timer: NodeJS.Timeout;
}

export class PluginSession {
  private active: WebSocket | null = null;
  private pending = new Map<string, Pending>();
  private waiters = new Set<() => void>();
  private readonly commandTimeoutMs: number;

  constructor(opts: { commandTimeoutMs?: number } = {}) {
    this.commandTimeoutMs = opts.commandTimeoutMs ?? 30_000;
  }

  isConnected(): boolean { return this.active !== null && this.active.readyState === 1; }

  setActive(ws: WebSocket) {
    if (this.active && this.active !== ws) {
      try { this.active.send(JSON.stringify({ type: "system", message: "Replaced by newer connection" })); } catch {}
      try { this.active.close(); } catch {}
      this.failAllPending(new Error("Plugin connection replaced"));
    }
    this.active = ws;
    ws.send(JSON.stringify({ type: "ready" }));
    this.notifyWaiters();
  }

  clearActive(ws: WebSocket) {
    if (this.active === ws) {
      this.active = null;
      this.failAllPending(new Error("Plugin disconnected"));
    }
  }

  handleResponse(msg: { id: string; status: "ok" | "error"; result?: unknown; error?: { message: string } }) {
    const p = this.pending.get(msg.id);
    if (!p) return;
    this.pending.delete(msg.id);
    clearTimeout(p.timer);
    if (msg.status === "ok") p.resolve(msg.result);
    else p.reject(new Error(msg.error?.message ?? "plugin error"));
  }

  async sendCommand(command: string, params: unknown): Promise<unknown> {
    if (!this.isConnected()) {
      await this.waitForConnection(3_000);
      if (!this.isConnected()) throw new Error("피그마 플러그인 연결을 확인해주세요");
    }
    const id = randomUUID();
    return new Promise<unknown>((resolve, reject) => {
      const timer = setTimeout(() => {
        this.pending.delete(id);
        reject(new Error(`Command ${command} timed out after ${this.commandTimeoutMs}ms`));
      }, this.commandTimeoutMs);
      this.pending.set(id, { resolve, reject, timer });
      try {
        this.active!.send(JSON.stringify({ type: "command", id, command, params }));
      } catch (e) {
        clearTimeout(timer);
        this.pending.delete(id);
        reject(e instanceof Error ? e : new Error(String(e)));
      }
    });
  }

  private waitForConnection(timeoutMs: number): Promise<void> {
    if (this.isConnected()) return Promise.resolve();
    return new Promise((resolve) => {
      const timer = setTimeout(() => {
        this.waiters.delete(waiter);
        resolve();
      }, timeoutMs);
      const waiter = () => { clearTimeout(timer); this.waiters.delete(waiter); resolve(); };
      this.waiters.add(waiter);
    });
  }

  private notifyWaiters() { for (const w of this.waiters) w(); }
  private failAllPending(err: Error) {
    for (const [id, p] of this.pending) {
      clearTimeout(p.timer);
      p.reject(err);
      this.pending.delete(id);
    }
  }
}
```

## 8. plugin-ws.ts — WS upgrade handler

```ts
import { WebSocketServer, type WebSocket } from "ws";
import type { Server as HttpServer } from "node:http";
import type { PluginSession } from "./sessions.js";

export function attachPluginWebSocket(httpServer: HttpServer, session: PluginSession): void {
  const wss = new WebSocketServer({ noServer: true });

  httpServer.on("upgrade", (req, socket, head) => {
    if (req.url === "/ws") {
      wss.handleUpgrade(req, socket, head, (ws) => wss.emit("connection", ws, req));
    } else {
      socket.destroy();
    }
  });

  wss.on("connection", (ws: WebSocket) => {
    session.setActive(ws);
    ws.on("message", (raw) => {
      let msg: any;
      try { msg = JSON.parse(raw.toString()); } catch { return; }
      if (msg?.type === "response") session.handleResponse(msg);
    });
    ws.on("close", () => session.clearActive(ws));
    ws.on("error", () => session.clearActive(ws));
  });
}
```

## 9. src/server/index.ts (개요)

```ts
import express from "express";
import { Server } from "@modelcontextprotocol/sdk/server/index.js";
import { SSEServerTransport } from "@modelcontextprotocol/sdk/server/sse.js";
import { CallToolRequestSchema, ListToolsRequestSchema } from "@modelcontextprotocol/sdk/types.js";
import { zodToJsonSchema } from "zod-to-json-schema";
import { PluginSession } from "./sessions.js";
import { attachPluginWebSocket } from "./plugin-ws.js";
import { TOOLS } from "./tools.js";

const PORT = Number(process.env.PORT ?? 3000);

const session = new PluginSession();
const mcp = new Server({ name: "figma-prototype-mcp", version: "0.18.0" }, { capabilities: { tools: {} } });

mcp.setRequestHandler(ListToolsRequestSchema, async () => ({
  tools: TOOLS.map(t => ({ name: t.name, description: t.description, inputSchema: zodToJsonSchema(t.schema) as any })),
}));

mcp.setRequestHandler(CallToolRequestSchema, async (req) => {
  const tool = TOOLS.find(t => t.name === req.params.name);
  if (!tool) return { isError: true, content: [{ type: "text", text: `Unknown tool: ${req.params.name}` }] };
  const parsed = tool.schema.safeParse(req.params.arguments ?? {});
  if (!parsed.success) return { isError: true, content: [{ type: "text", text: `Invalid input: ${parsed.error.message}` }] };
  try {
    const result = await session.sendCommand(tool.command, parsed.data);
    return { content: [{ type: "text", text: JSON.stringify(result, null, 2) }] };
  } catch (e) {
    return { isError: true, content: [{ type: "text", text: (e as Error).message }] };
  }
});

const app = express();
const transports = new Map<string, SSEServerTransport>();

app.get("/sse", async (_req, res) => {
  const transport = new SSEServerTransport("/messages", res);
  transports.set(transport.sessionId, transport);
  res.on("close", () => transports.delete(transport.sessionId));
  await mcp.connect(transport);
});

app.post("/messages", express.json(), async (req, res) => {
  const t = transports.get(String(req.query.sessionId ?? ""));
  if (!t) { res.status(400).send("unknown session"); return; }
  await t.handlePostMessage(req, res);
});

const httpServer = app.listen(PORT, () => {
  console.log(`[server] listening on http://localhost:${PORT}`);
  console.log(`[server]   MCP SSE endpoint: GET /sse`);
  console.log(`[server]   Plugin WebSocket:  ws://localhost:${PORT}/ws`);
});
attachPluginWebSocket(httpServer, session);
```
